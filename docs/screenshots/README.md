# Capturas de la landing

Coloca aquí los PNG y la galería de la web los mostrará automáticamente
(si falta alguno, esa tarjeta se oculta sola). Nombres esperados:

| Fichero | Qué mostrar |
|---|---|
| `perfil.png` | La vista de edición de un perfil (pestañas Conexión/Sincronización/Notificaciones). |
| `explorador.png` | La pestaña Explorador con ficheros remotos. |
| `ajustes.png` | El modal de Ajustes (idioma, tema, seguridad…). |
| `actividad.png` | La vista de monitorización / panel de Actividad con el watcher activo. |

Recomendación: ~1400 px de ancho, fondo de la app (tema oscuro o claro, consistente).
Captura la ventana de la app (en macOS, `Cmd+Shift+4` y luego `Espacio` para capturar la ventana).
