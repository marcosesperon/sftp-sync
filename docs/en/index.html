Features

Everything you expect from an SFTP sync, in a real app.

Built for development and deployments: fast, configurable and editor-independent.

Upload on save

A watcher monitors your folder and uploads changes instantly, smartly grouping the editor's save bursts.

Multi-profile

Manage several servers, duplicate them in one click and export/import your configuration any time.

Host key verification

TOFU model with its own known_hosts: confirm the fingerprint the first time and get alerted if the server key changes.

Remote explorer

Browse the remote tree, rename and delete, multi-select, and drag local files in to upload them.

SSH connection

Open an SSH terminal with the profile credentials: built into the app (multi-session), in the system terminal, or with PuTTY on Windows.

Native notifications

Per-profile system alerts: errors only, batch summary or every action. You decide the noise.

Background + tray

Close the window and keep watching from the system tray. Single instance, no duplicates.

Theme & language

Automatic light/dark based on the system (or forced), and Spanish / English UI.

Fine-grained rules

include/ignore patterns, mirror mode, empty folders and propagated deletes.

Key or password

Private-key (with passphrase) or password auth, with modern RSA support (rsa-sha2).

How it works

From zero to syncing in a minute.

Create a profile

Host, user and key or password. Pick the local folder and remote path with the native picker.

Test the connection

Verify the server key once. From then on, everything connects without friction.

Start the watcher

Save in your editor and changes travel on their own. Close it to the tray and forget about it.

Full control

Clear, per-profile configuration.

  • ✓include / ignore glob patterns to upload exactly what you want.
  • ✓Mirror mode to reflect deletions on the remote.
  • ✓Configurable autoDelete and empty folders.
  • ✓Resizable activity, commands and explorer panel.
profiles.json
"host": "10.10.10.1",
"port": 22,
"username": "deploy",
"localRoot": "~/web/app",
"remotePath": "/var/www/",
"include": ["**/*"],
"ignore": [".git", "node_modules/**"],
"uploadOnSave": true,
"autoDelete": false   // optional mirror mode
Security

Connections you can verify.

Pure-Rust core (no native C dependencies), OpenSSH-style host key verification and configuration that stays on your machine.

SSHSFTP over russh, with key or password authentication.
TOFUSHA256 fingerprint on trust and an alert if the server key changes.
LOCALNo telemetry. Your configuration never leaves your machine.
